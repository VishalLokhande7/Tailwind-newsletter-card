# 📖 About the Project

This project is a modern **Newsletter Subscription Card** built using **Tailwind CSS**. It focuses on creating a clean, responsive, and visually appealing UI using utility classes without writing custom CSS.

The project helped me practice layout design, responsive development, gradients, spacing, typography, flexbox, and Tailwind CSS utilities.

- 📩 Modern Newsletter Card UI
- 🎨 Beautiful Gradient Header
- 📱 Fully Responsive Design

# ⚙️ Getting Started

Clone the repository

```bash
git clone https://github.com/VishalLokhande7/tailwind-newsletter-card.git
```

Go to the project directory

```bash
cd tailwind-newsletter-card
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run start
```
